# Data Cleaning & Preparation using Excel

## Project Overview

This project was completed as part of the **DecodeLabs Data Analytics Internship - Project 1**. The objective was to clean a raw dataset using **Microsoft Excel** by identifying missing values, checking for duplicates, and verifying data formats.

## Dataset

* **Dataset:** E-commerce Orders Dataset
* **Records:** 1200
* **Columns:** 14

## Tasks Performed

* Identified missing values
* Filled missing coupon codes with "No Coupon"
* Checked for duplicate Order IDs
* Verified date formats
* Verified numeric and text formats
* Prepared a clean dataset for analysis

## Tools Used

* Microsoft Excel
* Advanced Excel
* Filters
* Conditional Formatting
* Excel Formulas

The dataset was successfully cleaned and verified, making it ready for further data analysis.
